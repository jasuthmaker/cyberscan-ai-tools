# CyberScan AI Suite

All 5 cybersecurity tools in a single premium web app.

## Tools
- **Password Analyzer** — strength score, character checks, improvement tips
- **IP Classifier** — IPv4/IPv6 classification, class, private/global flags
- **URL Analyzer** — risk scoring, suspicious TLD/IP/path detection
- **PII Scanner** — detect emails, phones, SSNs, credit cards, sensitive keywords
- **Hash & Encoding** — MD5/SHA/Blake2b hashing, Base64/Hex/URL encoding

## Run

```bash
pip install -r requirements.txt
python app.py
```

Open `http://localhost:8000`

## API Endpoints

```
POST /api/password   { "password": "..." }
POST /api/ip         { "ip": "..." }
POST /api/url        { "url": "..." }
POST /api/text       { "text": "..." }
POST /api/hash       { "text": "..." }
POST /api/encode     { "text": "..." }
POST /api/decode     { "encoded": "..." }
```

## Stack
- Python 3.10+ · FastAPI · Uvicorn
- Vanilla HTML/CSS/JS · Tailwind CSS CDN
- Core logic generated by AI agent loop (Manager → Developer → Tester)

## Deploy (Railway / Render / Fly.io)
Set start command: `uvicorn app:app --host 0.0.0.0 --port $PORT`
